# 心情日记

一个简单的日记记录站点，根据分类实现数据统计。


[演示地址](http://yinhengli.com:8081)  

技术栈：    
- react
- react-router
- axios 
- echarts
- mongodb
- antd-mobile 
  
脚手架工具:  
create-react-app  
  
运行方式  
1、yarn 启动  
  - yarn install
  - yarn run start （前端启动）
  - yarn run nodeserver (node端启动)

2、发布项目
  - yarn run publish
